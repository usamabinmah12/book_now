**BookNow** is an ultra-modern, high-performance travel destination and luxury accommodation booking application. Designed with user experience in mind, it allows travelers to explore exquisite locations worldwide, read authentic reviews, and securely book their dream vacations.

🌐 **Live Website:** [https://book-now-six.vercel.app](https://book-now-six.vercel.app)

---

## ✨ Features

- **🌍 Diverse Destinations:** Explore a curated collection of villas, tropical bamboo chalets, glass cabins, castles, and underwater suites.
- **🔐 Secure Authentication (Better-Auth):** Complete authentication setup including Email/Password and Google Social Login.
- **⚡ Super-fast Search & Filter:** Instantly filter destinations by category (Resort, Villa, Cabin, Glamping, Castle) and search by location.
- **🛡️ Role-Based Access Control:** Dynamic routing and personalized dashboards based on user roles (`user`, `creator`, `admin`).
- **📱 Seamless UI/UX:** Responsive mobile drawer, smooth animations with Framer Motion, and a dark-themed futuristic layout powered by HeroUI (NextUI) and TailwindCSS.
- **🔗 Relational Destination Queries:** Promotes smart discovery using matching `relatedItems` for each spot.

---

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router), TypeScript, Tailwind CSS, Framer Motion, HeroUI (formerly NextUI)
- **Backend & Database:** MongoDB (via MongoClient), Next.js Server Actions / Route Handlers
- **Authentication:** Better-Auth (with MongoDB adapter)
- **Icons:** Gravity UI Icons
